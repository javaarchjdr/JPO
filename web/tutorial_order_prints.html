Tutorial: Order prints from a lab

Objective:

In this tutorial you will learn:

  • How to order prints from from a lab that has a website with upload
  • How to copy the pictures onto a Memory Stick

Quick Take

Simply select the pictures you want printed and drag them to the web site of the lab.

Alternatively create a directory somewhere and copy the pictures you want to print there. User the Copy Picture entry in the the pop-up menu. The directory can be on a memory stick that you then send to the lab. Or it could be a directory that you the burn on a CD.
